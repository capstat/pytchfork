#README
Included in this folder is everything needed to run my 602 Final Project.
1) all_pitchfork_albums.csv
	all pitchfork reviews from 1999 till 12/7/15
2) Pitchfork_Analysis.ipynb
	python notebook file for pitchfork album data analysis
3) pitchfork_gatherer.py
	used to extract review metadata from pitchfork.com
4) pytchfork.py
	main file to run the program that lets user explore the pitchfork reviews
5)welcome.py
	used as a welcome message generator for pytchfork.py
